<h1>
Main Heading
<h2>
Level 2 Heading
<h3>
Level 3 Heading
<h4>
Level 4 Heading
<h5>
Level 5 Heading
<h6>
Level 6 Heading
<p>
Paragraph Tag
<b>
Makes characters appear BOLD
<i>
Makes characters appear in italic format
<sup>
Superscript tag for suffixes of dates
Example of the superscript tag is the TH in 4th
<sub>
Subscript tag for footnotes etc.
Example of the subscript tag is the 2 in H20
<br />
Inserts a line break
<hr />
Inserts a horizontal rule between sections
<strong>
Strong tag shows content as BOLD
<em>
Emphases tag shows content as Italicized
<blockquote>
Blockquote tag indents the characters of a quote in the content of the paragraph
<q>
Quote Tag is used for shorter quotes in a paragraph
<abbr> and <acronym>
Abbreviation tag is used to abbreviate content.
Example NASA
<cite>
Cite Tag can be used to indicate where the citation is from.
<dfn>
Defining tag is used to indicate the defining instance of a new term.
<address>
Address tag has quite a specific use: to contain contact details for the author of the page.
<ins />
Insert tag can be used to show content that has been inserted into a document
<del />
Delete tag can show text that has been deleted from it
<s />
Strikethrough Tag indicates something that is no longer accurate or relevant."Delete"
<ol> and <li>
Ordered List tag and List tag are used to create lists specified by the type of numbering.
<ul>
Unordered list tag creates a list without specific numbering, only bulleted layout.
<li>
Each item in the list is placed between an opening and closing List tag.
<dl>
Definition List tag creates a list which consists of a series of terms and their definitions or etc.
<dt>
Definition Term tag is used to contain the term being defined"
<dd>
"Different Definitions tag is used when there are two terms used for the same definition or two different definitions for the same term.
